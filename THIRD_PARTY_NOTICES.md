## Updating BMAD files

If BMAD files need to be updated, **rerun the official BMAD installer** rather than manually editing the generated core files:

```bash
npx bmad-method@latest install --action update --tools claude-code --yes
```

Manually editing files under `_bmad/` or `.claude/skills/bmad-*` is discouraged because the next installer run regenerates them.
